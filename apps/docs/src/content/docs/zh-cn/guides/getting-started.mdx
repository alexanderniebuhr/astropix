指南引导用户完成他们想要完成的特定任务，通常包含一系列步骤。
编写好的指南需要考虑用户试图做什么。

:::tip[你知道吗？]
Starlight 是一个使用 [Astro](https://astro.build/) 构建的文档网站工具包。你可以用以下命令开始：

```sh
npm create astro@latest -- --template starlight
```
:::
