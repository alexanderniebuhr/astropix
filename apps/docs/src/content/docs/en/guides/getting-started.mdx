## Usage

Astropix is a Integration for Astro projects. To use it, you need to add it in your project config:

```js ins={3,6-8}
// astro.config.mjs
import { defineConfig } from 'astro/config';
import astropix from "astropix";

export default defineConfig({
  integrations: [
    astropix()
  ]
})
```
